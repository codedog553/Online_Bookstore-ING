# 前端視覺美化說明（藍白主題）

## 美化重點
- 建立藍白主題配色與全域字體，強化標題層級與閱讀對比。
- 導覽列與頁面骨架加上柔和陰影與邊界，提升層次但保留簡潔風格。
- 卡片、表格、表單元件統一圓角與淡藍底色，維持一致性。
- 商品卡片與詳情區塊加入細緻邊框與陰影，價格與重點資訊以藍色強調。
- 管理報表圖表色系改為藍色階，視覺與主題一致。
